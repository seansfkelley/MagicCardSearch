# Mana Symbol Images

This directory should contain the mana symbol images for Magic: The Gathering cards.

## Required Images

Create PNG files (ideally 64x64 or 128x128 pixels) for the following mana symbols:

### Basic Mana Colors
- `mana_w.png` - White mana
- `mana_u.png` - Blue mana
- `mana_b.png` - Black mana
- `mana_r.png` - Red mana
- `mana_g.png` - Green mana
- `mana_c.png` - Colorless mana

### Generic/Numeric Mana
- `mana_0.png` through `mana_16.png`
- `mana_20.png`
- `mana_x.png`, `mana_y.png`, `mana_z.png`

### Hybrid Mana
- `mana_wu.png` - White/Blue
- `mana_wb.png` - White/Black
- `mana_ub.png` - Blue/Black
- `mana_ur.png` - Blue/Red
- `mana_br.png` - Black/Red
- `mana_bg.png` - Black/Green
- `mana_rw.png` - Red/White
- `mana_rg.png` - Red/Green
- `mana_gw.png` - Green/White
- `mana_gu.png` - Green/Blue

### Phyrexian Mana
- `mana_wp.png` - White Phyrexian
- `mana_up.png` - Blue Phyrexian
- `mana_bp.png` - Black Phyrexian
- `mana_rp.png` - Red Phyrexian
- `mana_gp.png` - Green Phyrexian
- `mana_p.png` - Phyrexian

### Hybrid Generic/Colored
- `mana_2w.png` - 2 or White
- `mana_2u.png` - 2 or Blue
- `mana_2b.png` - 2 or Black
- `mana_2r.png` - 2 or Red
- `mana_2g.png` - 2 or Green

### Special Symbols
- `mana_s.png` - Snow
- `mana_tap.png` - Tap symbol
- `mana_untap.png` - Untap symbol
- `mana_e.png` - Energy
- `mana_chaos.png` - Chaos symbol

## Resources

You can download official mana symbols from:
- Scryfall's asset repository
- MTG JSON icon assets
- Community-created symbol sets

Make sure to respect copyright and licensing terms when using mana symbols.
